# baia-vision-poc — demo

Front de teste da POC de visão computacional para baia de carga: detecta
caminhão e pessoas, cronometra a operação e emite alertas. Suba um vídeo
royalty-free (Pexels/Pixabay/Mixkit) e veja a tabela de eventos + o vídeo
anotado.

> POC para demonstração — não é sistema de segurança de vida, não identifica
> pessoas e não lê placas. Código: https://github.com/AliceCullen-html/ReconhecimentoPy
